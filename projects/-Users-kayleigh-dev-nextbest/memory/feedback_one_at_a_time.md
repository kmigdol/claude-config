---
name: feedback_one_at_a_time
description: Present acceptance criteria and decisions to the user one at a time, not batched
type: feedback
---

When multiple agents report back with acceptance criteria or questions, present them to the user one at a time — not all at once in a batch.

**Why:** Batching multiple decisions overwhelms and makes it hard to focus on each one. The user prefers to work through them sequentially.

**How to apply:** When multiple agents send acceptance criteria simultaneously, present the first one, wait for the user's response, then present the next. Same for any multi-item decision list.
